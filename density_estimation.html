Tasks¶

  1. The task is to implement density estimation using the K-NN method. Obtain an iid sample of N1 points from a univariate normal (Gaussian) distribution (let us call the random variable X) centered at 1 and with variance 2. Now, empirically obtain an estimate of the density from the sample points using the K-NN method, for any value of K, where 1 KN. Produce one plot for each of the following cases (each plot should show the following three items: the N data points (instances or realizations of X) and the true and estimated densities versus x for a large number e.g., 1000, 10000 of discrete, linearly-spaced x values): (i) K = N = 1, (ii) K = 2, N = 10, (iii) K = 10, N = 10, (iv) K = 10, N= 1000, (v) K = 100, N= 1000, (vi) K = N = 50,000. Please provide appropriate axis labels and legends. Thus there should be a total of six figures (plots), each figure similar to Fig. 8 on p. 20 of the pdf notes.
  2. Re-estimate the density of Q1 using the kernel method (use Gaussian kernel), instead of the KNN method. That is, first obtain an iid sample of N1 points from the true density, and then obtain, by Gaussian KDE, an estimate of the true density using those N sample points. Produce plots (each plot showing the data points and the true and the estimated densities), one plot for each of the following sample sizes (note: sample size = N): 1, 2, 5, 10, 50, 100, 1000. A total of seven plots are to be produced for this part.
  3. Repeat Q2 by replacing the KDE method with the histogram method. Show a plot of the histogram, separately for three sample sizes 5, 10, 500 with 2, 5, and 20 bins used for each sample size. Thus a total of nine histograms will be produced.

Q1 - Estimation by K-NN¶

Creating the KNN_Estimator Class¶

In [3]:
class KNN_Estimator:
    def __init__(self, X, x_norm, K, N):
        self.X = X
        self.x_norm = x_norm
        self.K = K
        self.N = N
    
    def KNN_Density_Estimation(self):
        densities = np.zeros(self.x_norm.shape)
        # calculate radii and densities
        for i in range(1000):
            radius = np.abs(self.X['x'] - self.x_norm[i]).sort_values()[:self.K+1].max()
            densities[i] = self.K / (2 * radius * self.N)
        return densities

Q2 - Gaussian KDE¶

Creating the G_KDE Class¶

In [5]:
class G_KDE:
    def __init__(self, X, x_norm, N, sigma):
        self.X = X
        self.x_norm = x_norm
        self.N = N
        self.sigma = sigma
    
    # returns estimated densities
    def Gaussian_KDE(self):
        densities = np.zeros(x_norm.shape)
        for i in range(1000):
            densities[i] = self.Calc_Gauss_Fhat(self.x_norm[i], self.X['x'], self.N, self.sigma)
        return densities
    
    # calculates estimated density
    def Calc_Gauss_Fhat(self, x, X_j, N, sigma):
        rho = self.Gaussian_Kernel(x, X_j, sigma)
        return np.sum(rho) / (N * sigma)
    
    # calculates the gauss kernel
    def Gaussian_Kernel(self, x, X_j, sigma):
        X = np.full(X_j.shape, x)
        frac = 1 / math.sqrt(2 * math.pi)
        exp = np.exp(-0.5 * ((X - X_j) / sigma) ** 2)
        return frac * exp

Q3 - Histogram Method¶

Creating the Histogram Class¶

In [7]:
class Histogram:
    def __init__(self, X, num_bins):
        self.X = X
        self.h = (X.max() - X.min()) / num_bins
        self.bins = np.arange(X.min(), X.max()+self.h, self.h)
        self.bins[0] -= .0001
        
    # returns estimated densities of bins
    def calc_densities(self):
        count_in_bins = self.X.groupby(pd.cut(self.X, self.bins)).count()
        return count_in_bins / (self.h * self.X.count())
